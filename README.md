# **University_Practice_Autumn_2025**
In this repository will be presented my practice in the MISIS University
## **Введение**
Учебная практика в НИТУ МИСИС была разделена на несколько задач:
* Научиться работать с лазерным станком
Проектирование (CAD) новогоднего набора-конструктора под лазерную резку из фанеры толщиной 3 мм. 
* Научиться работать с 3D-принтерами
Изготовление 3D-печатных инструментов из PLA/PETG
* Улучшить рабочее пространство в FabLab
Уборка пространства, снятие прошлой стены, установка на её месте новой
* Создать органайзеры для инструментов
Создание концептов органайзеров
## **Работа с лазерным станком**
**Описание задачи:**
Освоение работы с лазерным станком и применение полученных знаний для проектирования и изготовления новогоднего набора-конструктора из фанеры толщиной 3 мм. Проект требовал комплексного подхода: от создания 3D-модели в CAD-программе до практической реализации на лазерном станке.
**Процесс выполнения:**
В ходе работы нужно было изучить работу лазерного станка, включая настройку мощности лазера, скорости перемещения, фокусировки и параметров для различных материалов. Особое внимание уделил безопасности при работе с оборудованием.
На создание модели для резки меня вдохновили снежинки, которые вырезаются из бумаги (Рисунок 1).

## **Работа с 3D-принтерами и изготовление инструментов**
**Описание задачи:**
Второй задачей практики было освоение технологии 3D-печати и применение её для изготовления специализированных инструментов из PLA и PETG пластиков. Целью было создание функциональных инструментов, которые могли бы быть использованы в повседневной работе FabLab.

**Процесс выполнения:**
В ходе работы нужно было изучить особенности работы с 3D-принтерами Prusa mk3, научиться работать со слайсером PrusaSliser.
